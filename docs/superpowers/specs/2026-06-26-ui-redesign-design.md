## 1. 配色方案

全部通过修改 `css/styles.css` 中 `:root` CSS 变量实现。

| 变量 | 旧值 | 新值 | 用途 |
|------|------|------|------|
| `--ai-color-bg` | `#0a0e27` | `#060d1f` | 全局背景（更深的蓝黑底，全页面统一） |
| `--ai-color-card` | `#111634` | `rgba(16,30,70,0.5)` | 卡片背景（半透明深蓝 + 毛玻璃） |
| `--ai-color-border` | `#1e2a4a` | `rgba(30,80,180,0.2)` | 卡片边框（微弱蓝光） |
| `--ai-color-cyan` | `#00d4ff` | `#1a5cff` | 主强调色（更亮的蓝） |
| `--ai-color-green` | `#10b981` | `#0ed9b4` | 成功/正常标识（青绿） |
| `--ai-color-red` | `#ef4444` | `#ff3b5c` | 异常/高风险（亮红） |
| `--ai-color-orange` | `#f59e0b` | `#f5a623` | 警告/中风险 |
| `--ai-color-text-primary` | `#e2e8f0` | `#e8edf5` | 主文字（偏白） |
| `--ai-color-text-secondary` | `#94a3b8` | `#7b8fb5` | 次要文字（淡蓝灰） |

### 新增变量

- `--ai-glow-blue`: `0 0 16px rgba(26,92,255,0.3)` — 卡片/标题蓝光阴影
- `--ai-glow-card`: `0 0 20px rgba(26,92,255,0.06)` — 卡片弱光边框阴影

### 全局背景统一

- 全页面（含 header、所有面板卡片）统一使用 `#060d1f` 背景色
- 卡片使用 `backdrop-filter: blur(8px)` 半透明毛玻璃效果
- 所有卡片边框统一使用 `rgba(30,80,180,0.2)`，带微弱蓝光

## 3. 照片处理 — 流式输出 + 粒子扫描效果

**文件**: `js/app.js` + `css/styles.css`（不修改 `js/analysis-engine.js` 内部逻辑）

### 3.1 流式输出

- 利用现有 `_onImageUpdate` / `_onStageChange` 回调
- **逐张 + 逐阶段流式**：每张图片每完成一个阶段，立即在 UI 中追加/更新该图片的处理卡片
- 每张图片渲染为独立的"处理卡片"：
  - 左侧：缩略图（带扫描线动画）
  - 右侧：图片名 + 三个阶段的实时状态
    - ✓ 完成（绿色 `#0ed9b4`）
    - ⏳ 进行中（蓝色 `#1a5cff` + 脉冲动画）
    - ○ 等待中（灰色 `#3a4f70`）
- 待处理图片显示为虚线边框占位卡片

### 3.2 粒子扫描效果

在处理中图片的缩略图上叠加一个 `<canvas>` 元素，绘制扫描线动画：

- **扫描线**: 一条淡蓝色柔和水平实线，颜色 `rgba(100,160,255, 0.35~0.5)`
- **渐变**: 两端渐隐淡出（`transparent → rgba(100,160,255,0.35) → rgba(130,180,255,0.5) → rgba(100,160,255,0.35) → transparent`）
- **光晕**: 扫描线上方 20px 微弱蓝光渐变带，透明度仅 0.04
- **运动**: 从上到下匀速移动，循环往复（类似安检机扫描）
- **四角定位线**: 图片四角各 10px 蓝色边框标记，透明度 0.25
- **底部状态栏**: 显示"⏳ 处理中..."和进度百分比
- Canvas 2D 实现，使用 `requestAnimationFrame` 驱动动画

## 4. 审计分析报告 — 弹窗展示

**文件**: `audit_agent.html` + `js/app.js`

### 4.1 弹窗容器

- 分析完成后，不再在原页面内嵌展示报告
- 将报告渲染至弹窗（复用/改造现有 `#aiResultModal` 或新增）
- 弹窗样式：
  - 半透明深蓝背景 `rgba(6,13,31,0.85)` + `backdrop-filter: blur(12px)`
  - 圆角 12px，边框 `rgba(30,80,180,0.3)`
  - 顶部标题栏：📊 审计分析报告 + 日期信息 + ✕ 关闭按钮
  - 底部：📥 导出报告 + 关闭 按钮

### 4.2 弹窗内容

**内容与原有报告完全一致，不做修改**：
- 4 列统计卡片（风险项 / 关注项 / 合规 / 已识别）
- AI 智能分析文字（紫色调区域）
- 原有报告渲染逻辑保持不变，仅改变渲染目标容器（从页面内嵌 → 弹窗内）

### 4.3 关闭方式

- 点击右上角 ✕ 按钮关闭
- 点击底部"关闭"按钮关闭
- 点击弹窗遮罩层关闭（可选）
- 关闭后回到主界面，分析结果不丢失
